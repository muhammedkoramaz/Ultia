## Project Description
A desktop application that performs the operations of adding a product to the warehouse, leaving the warehouse, assigning it to the team or personnel, and then selling it.

## Tech Stack
- C#
- Windows Form
- N-tier Architecture 
- MS-SQL
- ADO.NET
