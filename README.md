# Varlık - Zimmet Projesi
## Proje Açıklaması
Bir ürünün depoya eklenmesi, depodan çıkıp ekip veya personele zimmetlenmesi ve daha sonrasında satılması işlemlerini yapan bir masaüstü uygulaması.

## Kullanılan Teknolojiler 
- C#
- Windows Form
- N-tier Architecture 
- MS-SQL
- ADO.NET
